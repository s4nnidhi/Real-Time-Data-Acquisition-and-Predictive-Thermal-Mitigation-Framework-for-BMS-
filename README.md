# Real-Time-Data-Acquisition-and-Predictive-Thermal-Mitigation-Framework-for-BMS-
Predictive thermal mitigation system for 3S2P  Li-ion battery packs. Detects thermal runaway  20s early using dT/dt analysis, autonomously  derates overheating branches via PWM throttling,  and serves a real-time web dashboard over Wi-Fi.
